# Installation

This library is based on React!  
Please make sure that you have installed React 17+ in your project.

Now install with one of the following commands:

```bash title="NPM"
  npm install @justinlu/web-ui
```
```bash title="Yarn"
  yarn add @justinlu/web-ui
```
